## 2. 冷知识

- HTML 文件的扩展名是 `.htm/.html`

  > 因历史遗留问题，Win95/Win98 系统的文件扩展名不能超过 3 个字符，所以使用 `.htm`

- HTML 元素是不区分大小写的

  > <A></A> 等同于 <a></a>
